## ⚙️ Installation

**1. Clone the repository**

**2. Install server dependencies:**
  - 'cd server'
  - 'npm install'

**3. Install client dependencies:**
  - 'cd client'
  - 'npm install'


**Running the Application**

Start the server:
  - 'cd server'
  - 'npm start'

Start the client:
  - 'cd client'
  - 'npm run dev'
